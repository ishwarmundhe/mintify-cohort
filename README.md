# Mintlify Landing Page Clone

A responsive, pixel-perfect recreation of the Mintlify landing page built with HTML and CSS. This project focuses on replicating the modern UI structure, bento-grid layouts, and precise styling of the original site.

## üìÇ Project Overview

This assignment involves recreating the major sections of the Mintlify marketing site, ensuring fidelity in spacing, typography, and color hierarchy. The codebase is structured to be clean, semantic, and easily maintainable.

### Recreated Sections

- **Header**: Full navigation bar including the logo, links (Resources, Documentation, Customers, etc.), and distinct action buttons.
- **Hero Section**: High-impact intro featuring the "New" announcement pill, primary CTA, email capture form, and a floating UI preview element.
- **Logo Garden**: A responsive grid displaying partner and client logos.
- **Features Grid**: A complex bento-grid layout highlighting key product capabilities (AI Agents, Knowledge Management, Assistant).
- **Enterprise Section**: Dedicated area for enterprise features, including a customer success story card (Anthropic) and security compliance badges.
- **Customers Carousel**: A testimonial section showcasing client success stories with card-based layouts.
- **CTA Footer**: Final call-to-action block with resource links for pricing and quick starts.
- **Footer**: Comprehensive footer with columnar navigation, social links, status indicators, and theme toggles.

## üé® Design System

### Typography

The project uses a modern sans-serif stack to match the clean aesthetic of the original site:
`Segoe UI`, `Tahoma`, `Geneva`, `Verdana`, `sans-serif`.

### Color Palette

The design relies on a dark-themed palette dominated by deep greys and vibrant greens.

| Color             | Hex                        | Usage                          |
| :---------------- | :------------------------- | :----------------------------- |
| **Primary Text**  | `#ffffff`                  | Main headings and body text    |
| **Background**    | `#08090a`                  | Main page background           |
| **Mint Green**    | `#18E299`                  | Primary logo accent            |
| **Deep Green**    | `#0C8C5E`                  | Secondary logo accent          |
| **Vibrant Green** | `#10b981`                  | Icons, tags, and status dots   |
| **Dark Blue**     | `#020617`                  | Text on light buttons          |
| **Glass White**   | `rgba(255, 255, 255, 0.1)` | Borders and card backgrounds   |
| **Muted Grey**    | `#94a3b8`                  | Subtitles and descriptive text |
